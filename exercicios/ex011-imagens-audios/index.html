Reproduzindo áudios

Gustavo Guanabara fala sobre ter certeza que o audio vai funcionar. a tag audio padrão não funcionaou no navegador dele por exemplo. A outra maneira é tag source dentro da tag audio

A tag para audio é source src. Diferentemente da source media type da imagem. Nessa ordem de source se o mp3 não funcionar no site, o navegador vai tentar o audio de baixo. Se não funcionar é porque o navegador não tem suporte para esses formatos de arquivos e ai deve aparecer o último parágrafo com o link.

Sobre o preload. Se colocar auto, o site só vai finalizar o carregamento do site inteiro depois de baixar o áudio e isso é perigoso se der alguma falha ou o arquivo for pesado. metadata é para carregar apenas as informações do arquivo como título e link. none não carrega nada até o usuário clicar no link

controls aparece a opção de play pause e volume

autoplay começa o audio automaticamente

loop inicia o audio novamente quando o audio terminar
